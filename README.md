# Workshop de Kubernetes

### Gracias por pasar por el Workshop!

En este workshop vamos a usar Kubernetes en IBM Cloud para realizar diferentes tareas como

* Desplegar un único POD con Ubuntu
* Desplegar Grafana para monitoreo
* Desplegar PHP y MongoDB integrado
* Desplegar una Node app en Docker
* Desplegar Wordpress completo

### Background

Tuve la oportunidad de crear este Workshop en el marco del Programa de Desarrollo de Contenido en Español, enfocado en IBM Cloud y Kubernetes, y que fuera publicado en el sitio de [IBM Developer](https://developer.ibm.com/) en Septiembre del 2020

Se puede acceder en el siguiente link: [Workshop de Kubernetes](https://marcelorum.github.io/kubeshops/)
